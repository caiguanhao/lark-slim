# lark-upload-image

```
go get -v github.com/caiguanhao/lark-slim/cmd/lark-upload-image
```

Usage:

```
Usage: lark-upload-image [file ...] - upload images from files or stdin to lark
  -app-id string
        lark app id (you can also use env LARK_APP_ID)
  -app-secret string
        lark app secret (you can also use env LARK_APP_SECRET)
  -send string
        also send image message to open_id, user_id, email or chat_id
  -type string
        image type (message or avatar) (default "message")
```
